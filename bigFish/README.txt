bertFish is a script that demonstrates transfer-learning using BERT pre-trained model (BERT-Base, for minimal size) with binary classification. Using a 
dataset of fraudulent emails, and a set of internal emails from the Enron dataset. Both can be found on Kaggle. (Nigerian fraud dataset & enron dataset)

bertFish is the main file through which all the scripts run. The program has many dependencies, which will be reported if you try to run the script.
create a virtual environmnet using: python3 -m venv /path/to/new/virtual/environment
it most simple to place it in the home directory: python3 -m venv new_env

activate the environment using: source new_env/bin/activate

run: pip install -r requirements.txt to download dependencies
